# Lamlib

Lamlib is Lambda functions library.

## Requirements

- [Apex](http://apex.run/)

## Setup

Copy and set `project.json`:

```
$ cp project.sample.json project.json
$ vim project.json
```

## Functions

- [ses-forwarder](tree/master/functions/ses-forwarder): Mail forwarder from SES to another
